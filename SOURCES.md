# FXD Sources and Research

This file records important sources, standards, API references, research, patents, and architectural decisions used while building FXD.

## Source categories

- CAD and geometry-kernel documentation
- STEP and DXF standards
- SOLIDWORKS and other CAD APIs
- Fixture-design and manufacturing references
- GD&T and tolerance references
- Welding-process and distortion references
- Robot and cobot accessibility
- Standard fixture-component catalogs
- Academic papers and patent research

## Decision record format

For each major decision, record:

- decision
- alternatives considered
- reason selected
- risks
- date
- supporting references

No major engineering claim should be accepted solely because an AI suggested it.
