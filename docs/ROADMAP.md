# FXD Roadmap

This document describes product direction by release stage. Detailed executable work remains in `BACKLOG.md`.

## 0.1 — Technical proof

- choose a commercially viable geometry stack
- import synthetic STEP assemblies
- preserve hierarchy, transforms, and units
- prove topology, clearance, collision, Boolean, and export operations

## 0.5 — Engineer-assisted fixture concepts

- engineering annotations
- build orientation and datum strategy
- baseplates, supports, stops, pins, and locator primitives
- editable concept generation
- deterministic validation

## 1.0 — Industrial weld-fixture workflow

- ranked fixture concepts
- weld and operator accessibility
- standard tooling libraries
- STEP, DXF, BOM, assumptions, and setup-package export

## Future

- robot and cobot process envelopes
- engineer-correction learning
- CAD-specific connectors
- assembly and inspection fixtures
- broader industrial tooling applications
