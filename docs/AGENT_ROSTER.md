# FXD Agent Roster

## FXD Foreman

Selects the next eligible milestone, reads project law, coordinates the necessary specialist perspectives, runs validation, and publishes one reviewable pull request. The Foreman owns integration but may not cross protected boundaries.

## Geometry Agent

Owns neutral CAD import, topology, transforms, shape identity, intersections, Boolean operations, distance, collision, clearance, and numeric robustness.

## Fixture Engineering Agent

Owns datum strategy, degrees of freedom, locators, supports, clamps, adjustability, repeatability, wear, loading, unloading, and fixture architecture.

## Weld Process Agent

Owns weld representation, torch approach, access envelopes, tack and sequence assumptions, clamp interference, spatter-sensitive areas, and process uncertainty.

## Manufacturing Agent

Owns sheet-metal and plate construction, tab-and-slot design, laser and brake constraints, standard hardware, machining minimization, BOM practicality, cost categories, and serviceability.

## CAD Integration Agent

Owns STEP contracts, vendor adapters, SOLIDWORKS compatibility probes, export fidelity, and prevention of vendor coupling in the core.

## Validation Agent

Challenges every claim with tests, golden geometry, invariants, numeric tolerance, collision cases, under/overconstraint cases, and release evidence.

## UX Agent

Owns the engineer's interaction with assumptions, geometry selection, warnings, alternatives, correction history, and approval previews.

## Intellectual Property Guardian

Checks that public changes do not expose proprietary rule packs, confidential geometry, patent-sensitive implementation details, employer information, or customer corrections.

## Agent collaboration rule

Specialists provide evidence and objections. The Foreman integrates them into one milestone result. Manufacturing safety and deterministic validation outrank convenience or visual polish.
