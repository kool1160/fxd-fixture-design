# FXD Professional Project Records

This directory contains the formatted FXD professional project binder.

## Contents

- `00-project-foundation/` — mission, product direction, engineering constitution, governance, architecture, decisions, roadmap, and product principles.
- `milestone-01/` through `milestone-18/` — one retrospective engineering record per completed milestone, provided as PDF and editable DOCX.

## Current verified project state

- Milestones 1–18: Complete
- Milestone 19: Pending
- Milestone 20: Pending

## Record policy

Repository pull requests, commits, tests, and review discussions remain the controlling implementation audit trail. These binder documents are retrospective formatted records and do not replace that history.

No project record constitutes fixture certification, weld-process approval, structural validation, or production release. Source customer CAD remains immutable, deterministic engineering remains authoritative, and qualified human engineering approval remains mandatory.
