## Record policy

Repository pull requests, commits, tests, workflow runs, and review discussions remain the controlling implementation audit trail. These binder documents are retrospective formatted records and do not replace that history.

No project record constitutes fixture certification, weld-process approval, structural validation, manufacturability approval, or production release. Source customer CAD remains immutable, deterministic engineering remains authoritative, and qualified human engineering approval remains mandatory.
